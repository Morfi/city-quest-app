# Городские Квесты

**ФИО:** Гайд Иван Павлович  
**Группа:** МПБ-301-О-01

Мобильное приложение для ежедневных квестов по поиску и фотографированию интересных мест в городе. Приложение превращает прогулки по городу в увлекательную игру, где пользователи получают задания (квесты), связанные с поиском и фотографированием архитектурных объектов, уличного искусства и других достопримечательностей. Пользователи могут исследовать город, выполнять задания, делиться фотографиями в общей ленте и соревноваться с другими участниками в рейтинге.

**Для кого:** Для активных жителей города, которые любят исследовать новые места, фотографировать интересные объекты и соревноваться с другими участниками.

## Стек

- **Язык и фреймворк:**
  - TypeScript
  - React Native
  - Expo (~49.0.15)

- **Ключевые библиотеки:**
  - React (18.2.0)
  - @react-navigation/native (^6.1.9) — навигация между экранами
  - @react-navigation/bottom-tabs (^6.5.11) — нижние вкладки
  - react-native-screens (~3.22.0)
  - react-native-safe-area-context (4.6.3)
  - expo-image-picker (~14.3.2) — работа с камерой и галереей
  - expo-camera (~13.4.4) — доступ к камере устройства

- **Сервисы и интеграции:**
  - Expo Go — для разработки и тестирования на реальных устройствах

## Быстрый старт

```bash
npm install
npm start
```

Затем откройте приложение в Expo Go на вашем устройстве.
