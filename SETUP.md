# Инструкция по настройке

## Быстрый старт

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Для Android:
   - Установите приложение Expo Go на ваше Android устройство
   - Отсканируйте QR-код из терминала
   - Или запустите на эмуляторе: `npm run android`

## Создание иконок (опционально)

Если вы хотите добавить свои иконки, создайте следующие файлы в папке `assets/`:

- `icon.png` - 1024x1024px (иконка приложения)
- `splash.png` - 1242x2436px (экран загрузки)
- `adaptive-icon.png` - 1024x1024px (адаптивная иконка для Android)
- `favicon.png` - 48x48px (для веб-версии)

Для быстрого старта можно использовать онлайн-генераторы иконок или оставить стандартные иконки Expo.

## Разрешения

Приложение автоматически запросит разрешения на:
- Камеру (для фотографирования)
- Галерею (для выбора фотографий)

Эти разрешения настраиваются в `app.json`.
